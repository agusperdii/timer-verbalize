# GEC Debate Timer

A specialized web application for managing time during debate competitions. Features automatic audio notifications and visual color indicators to highlight important phases throughout each session.

## Main Features
- **Timer Mode** — Counts down from a set time to zero
- **Stopwatch Mode** — Counts up from zero with a set limit
- **Audio Signals** — Automated bell sounds for key milestones
- **Visual Indicators** — Background colors change to signal time phases
- **Minimalist View** — Toggle controls to show only the timer display

## Getting Started
- Open index.html in a web browser
- Choose between Timer or Stopwatch
- Input the desired duration
- Click Start to begin the session
- Use Hide Controls for a cleaner presentation

## Files
- index.html — Web structure
- script.js — Timer logic
- style.css — Application styling
- teng.mp3 and over.mp3 — Notification sounds
